## pandabrew

A TUI tool for selective file extraction and project documentation

### Synopsis

PandaBrew üêº‚òï

A high-performance, headless-first tool to extract codebases into a single
text file for LLM context. Features an interactive TUI with workspace
management and smart file filtering.

```
pandabrew [path] [flags]
```

### Options

```
      --headless        Run in headless mode without TUI
  -h, --help            help for pandabrew
      --output string   Output file path (default "project_extraction.txt")
      --root string     Project root directory (default ".")
```
